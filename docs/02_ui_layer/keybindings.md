# Keybindings

## Global (all views)

| Key | Action |
|-----|--------|
| `Ctrl+P` | Focus prompt bar |
| `Esc` | Back to previous view / unfocus prompt |
| `Ctrl+C` | Cancel current task |
| `Ctrl+Q` | Quit |

## View Switching

| Key | View |
|-----|------|
| `1` | Task Board |
| `2` | Plan View |
| `3` | Agents View |
| `4` | Diff Review |
| `5` | Logs View |
| `6` | Memory Trace |

## Navigation (within views)

| Key | Action |
|-----|--------|
| `j` / `↓` | Scroll down / next item |
| `k` / `↑` | Scroll up / previous item |
| `g` | Go to top |
| `G` | Go to bottom |
| `/` | Search within view |
| `n` | Next search match |
| `Enter` | Select / expand item |
| `Tab` | Next pane / section |

## Diff Review

| Key | Action |
|-----|--------|
| `a` | Approve all |
| `r` | Reject all |
| `e` | Edit (prompt modification) |
| `n` | Next file |
| `p` | Previous file |

## Mode Switching

| Key | Action |
|-----|--------|
| `F1` | Rush mode |
| `F2` | Smart mode |
| `F3` | Deep mode |
